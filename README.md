##### CLI

#### Установим последнюю версию pipenv:
* `pip install pipenv`

#### Cоздадим виртуальное окружение, указав версию интерпретатор (--three / --two  Python 3/2), установим зависимости из файла Pipfile (из текущей директории проекта):
* `pipenv install  --three`
* `pipenv install  --two`

#### Активируем виртуальное окружение:
* `pipenv shell`


#### Для запуска программы без параметров (Windows):
* `python main.py`

#### Справочное сообщение:
* `python main.py --help`

#### Исходная директория, по умолчанию директория в которой запущен скрипт:
* `python main.py -s "C:\some\название каталога с пробелом"`
* `python main.py --src-dir=C:\some\название_каталога_без_пробела`

#### Целевая директория, по умолчанию директория в которой запущен скрипт:
* `python main.py -d "C:\some\название каталога с пробелом"`
* `python main.py --dst-dir=C:\some\название каталога с пробелом`
